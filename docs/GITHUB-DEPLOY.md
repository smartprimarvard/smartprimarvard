# Ersätt Platform 1.0 på GitHub

## Rekommenderat: GitHub Desktop

1. Packa upp ZIP-filen.
2. Klona eller öppna `smartprimarvard/smartprimarvard` i GitHub Desktop.
3. Öppna repositorymappen i Finder.
4. Radera de gamla filerna, men inte den dolda `.git`-mappen.
5. Kopiera in alla filer och mappar från Platform 2.0.0.
6. Commit: `Publicera Platform 2.0.0`.
7. Push till `main`.
8. Vänta på GitHub Pages-byggningen och ladda om webbplatsen med hård omladdning.

## Webbläsare

Välj **Add file → Upload files** och ladda upp innehållet från den uppackade ZIP-filen. Filer med samma namn ersätts. Nya filer läggs till. `om.html` omdirigerar äldre länkar till den nya bakgrundssidan.
